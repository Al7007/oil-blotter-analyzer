# Анализ капельного теста масла

Программа для **цифрового анализа капельного (blotter spot) теста** моторного масла по логике ASTM D7899. Загрузите фото пятна на фильтровальной бумаге — приложение выделит зоны хроматограммы **C / A / D / T**, построит спектрограмму и рассчитает ключевые показатели состояния масла.

**Разработчик:** Alim Unagasov

## Для чего

Полевой скрининг состояния масла без лаборатории:

- **DS** — коэффициент дисперсии (насколько сажа размазана по пятну)
- **MD** — индекс качества диспергирования (0–100)
- **CI** — загрузка сажой
- **Blotter-код** — трёхзначный качественный код (сажа / топливо / вода)
- Оценка **топлива** и **воды** (ориентировочно)
- **Окисление**, **симметрия**, **согласованность** нескольких капель на одном снимке
- Отчёт **«Потеря свойств»** — оценка деградации масла

> Точные значения топлива и воды требуют лаборатории (GC, Karl Fischer). Программа предназначена для быстрой оценки на месте.

## Как пользоваться

### Запуск (Windows)

1. Скачайте или клонируйте проект.
2. Дважды щёлкните **`run.bat`** или **`Запуск.bat`**.

При **первом запуске** скрипт сам установит Python и библиотеки (если их нет) и покажет в консоли, что скачивается. Нужен интернет.

### Работа в программе

1. **Вставить (Ctrl+V)** — вставить снимок из буфера  
2. **Открыть файл** — загрузить фото с диска  
3. **Веб-камера** — снять пятно с камеры  
4. **Анализ** — запустить обработку  
5. При нескольких каплях на снимке кликните по нужной капле  

**Кнопки отчёта:**

| Кнопка | Что показывает |
|--------|----------------|
| *(по умолчанию)* | Краткая диагностика |
| **Подробно** | Полный отчёт с пояснениями |
| **Описать результат** | Текстовый вывод и рекомендация |
| **Потеря свойств** | Оценка сохранности свойств масла |

**Цвета:** зелёный — норма, жёлтый — погранично, красный — опасно. **Клик по красному значению** — причина и рекомендация.

**Сохранить спектр / оверлей** — экспорт графика и разметки зон.

## Требования

- Windows 10/11
- Интернет — только при первой установке Python и пакетов
- Python 3.10+ (устанавливается автоматически через `run.bat`)

## Ручная установка (для разработчиков)

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Структура проекта

```
main.py          — точка входа
app.py           — графический интерфейс (tkinter)
pipeline/        — обработка изображения и расчёт метрик
bootstrap.ps1    — автоустановка Python и зависимостей
run.bat          — запуск для пользователя
```
